# Bash Note

**List all envrionment variables**
```shell
$ env
```

**Get a variable**
```shell
$ echo $varname
```

**Set a variable**
```shell
$ export varname=value
```

**Unset http proxies (LOCAL)**
```shell
$ unset http_proxy
$ unset https_proxy
$ unset HTTP_PROXY
$ unset HTTPS_PROXY
```
